создание бд и пользователя
    подключение к PostgreSQL как администратор
        sudo -u postgres psql

    в открывщейся консольке пишу:
        CREATE DATABASE quantum_experiments;
    пользователь для программы
        CREATE USER quantum_user WITH PASSWORD 'quantum_password';
    дача пользователю права на базу
        GRANT ALL PRIVILEGES ON DATABASE quantum_experiments TO quantum_user;
    выход из консоли PostgreSQL
        \q

    как удалить всё, если что-то пойдёт не так:
        Удалить базу данных:
            sudo -u postgres psql -c "DROP DATABASE quantum_experiments;"
        Удалить пользователя:
            sudo -u postgres psql -c "DROP USER quantum_user;"
        Удалить PostgreSQL полностью:
            sudo apt remove --purge postgresql postgresql-contrib
            sudo apt autoremove

    чтобы остановить службу PostgreSQL (также вынесет все дочерние процессы (в htop видно)): 
        sudo systemctl stop postgresql
    проверить статус:
        sudo systemctl status postgresql
        -- теперь сервер PostgreSQL перестаёт принимать подключения. Python будет получать ошибку подключения.
    как еще проверить:
        sudo -u postgres psql 
    для запуска:
        sudo systemctl start postgresql
    чтобы PostgreSQL запускался автоматически при загрузке системы:
        sudo systemctl enable postgresql
    чтобы PostgreSQL НЕ запускался автоматически при загрузке системы:
        sudo systemctl disable postgresql
    команды postgresql (вышеперечисленные):
        https://stackoverflow.com/questions/41528530/stopping-postgresql-from-starting-on-ubuntu-startup

    как удалять записи из БД с помощью DataGrip:
        1 Откройте таблицу в редакторе данных: В окне Database Explorer найдите вашу таблицу experiments, дважды щелкните по ней или нажмите F4.
        2 Выберите строки для удаления: Щелкните на номера строк (в серой области слева) или удерживайте Ctrl, чтобы выбрать несколько строк.
        3 Удалите строки:
            Нажмите кнопку Delete Row(s) на панели инструментов.
            Или используйте горячие клавиши Ctrl+Y или Delete.
        4 Сохранение изменений: После удаления строк изменения находятся в локальной памяти DataGrip. 
            Чтобы они вступили в силу в базе данных, нажмите кнопку Submit (Ctrl+Enter).
